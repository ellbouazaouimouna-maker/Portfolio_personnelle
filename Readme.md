#  Portfolio Personnel - Mouna El Bouazaoui

Ce projet est un **portfolio personnel moderne et interactif** développé pour présenter mes compétences, projets et expériences en tant qu’étudiante en ingénierie informatique.

---

##  Aperçu

Le site comprend :
- Une section **Hero** avec présentation personnelle
- Des **cartes de projets** avec effet 3D
- Une section **compétences et services**
- Une page **contact interactive**
- Une horloge en temps réel
- Un **curseur personnalisé**
- Un design moderne avec effets glassmorphism et animations

---

## 🛠️ Technologies utilisées

- HTML5
- CSS3 (Flexbox, animations, effets 3D)
- JavaScript (DOM, horloge dynamique)
- Font Awesome (icônes)

---

##  Structure du projet
```bash
portfolio/
│
├── index.html # Structure principale du site
├── style.css # Styles et animations
├── script.js # Scripts JS (horloge, curseur, etc.)
├── image.jpeg # Photo de profil
├── favicon.png # Icône du site
└── cv.pdf # CV téléchargeable
```

##  Fonctionnalités

-  Interface responsive et moderne
-  Curseur personnalisé
-  Horloge en temps réel
